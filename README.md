<h1>DEMIX - Спортивная обувь</h1>
<h2>О Проекте:</h2>
<p>Коммерческий проект предназначен для демонстрации навыков использования технологий, на стеке: JS, React (Context), NextJS, TS, MongoDB, Mongoose, Tailwindcss.</p>
